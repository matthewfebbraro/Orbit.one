# ⟐Orbit.one

**⟐Sync.one's Satellite Step Tracking App**  
Step Tracker that creates daily step data for use within ⟐Sync.one: Adaptive Metablic Interface
---

### Features
- Normal 12am Daily Reset
- NightHawk Mode 3am Daily Reset
- Enable and disable tracking button
- Manual Reset Button.
- No online usage, fully offline.
- Uses Androids built in step tracker to track steps
- small app size 11MB
  
---

## Tech Stack
- Kotlin (Android Studio)
- Custom-built StepService Script
